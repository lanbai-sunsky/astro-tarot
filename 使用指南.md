# 星罗 · 占星塔罗 — 使用指南

## 开始之前

**无需注册、无需登录、无需付费。** 所有数据保存在你的浏览器里，不会上传到任何服务器。

---

## 一、塔罗占卜

### 操作流程

1. **选择牌阵** — 点击「单张牌」或「三张牌」
   - 单张牌：适合每日指引、快问快答
   - 三张牌：过去 → 现在 → 未来，适合深入问题

2. **输入问题（可选）** — 心里默念或输入你想问的事，留空也可以

3. **开始抽牌** — 点击按钮，系统随机洗牌并发牌

4. **翻牌** — 逐个点击牌背，牌面会以 3D 翻转动画揭开
   - 牌面显示：牌名、正位/逆位、关键词
   - 逆位牌以特殊样式标注

5. **查看解读** — 全部翻开后显示完整解读，包含综合、感情、事业三方面

### AI 增强解读

- 点击「AI 增强解读」可获得更深度的个性化解读
- **需要先在设置页填入你的 DeepSeek API Key**（百万 Token 仅 ¥1）
- Key 仅保存在你的浏览器中
- 如 API 调用失败，不影响本地解读的正常使用

### 保存与历史

- 点击「保存」将结果存入历史
- 在「历史」页可随时回顾，最多保留 50 条
- 退出页面后当前占卜状态会清空

## 二、占星星盘

### 录入出生信息

打开星盘页面，按 3 步填入：

| 步骤 | 内容 | 说明 |
|------|------|------|
| 第 1 步 | 出生日期 | 选择年月日 |
| 第 2 步 | 出生时间 | 尽量精确到分钟；不确定可留空（默认 12:00） |
| 第 3 步 | 出生地点 | 输入城市名（如「北京」），时区会自动检测 |

系统内置了 20 个常见城市（北京、上海、东京、纽约、伦敦等）。输入其他城市会自动估算坐标。

### 星盘图与解读

提交后生成：

- **星盘圆图** — Canvas 绘制，包含：
  - 外层 12 星座彩色扇区
  - 中层 12 宫位线
  - 内层行星符号（逆行行星以红色标注）
  - 相位连线（不同颜色代表不同相位）

- **解读面板** — 从上到下依次展开：
  - **上升星座** — 醒目展示，附带星座特质解读
  - **行星位置** — 10 颗行星各自所在的星座和宫位
  - **主要相位** — 合、冲、拱、刑、六合相位列表
  - **元素与模式** — 火/土/风/水元素统计 + 基本/固定/变动模式

### 修改出生信息

点击星盘上方的「修改」按钮，可重新编辑出生信息并自动重新计算。

---

## 三、历史记录

- 塔罗和星盘分两个 Tab 独立展示
- 每条记录显示时间戳和简要信息
- 点击卡片可查看完整解读内容
- 支持删除单条记录

---

## 四、设置

| 设置项 | 说明 |
|--------|------|
| 出生信息 | 查看和编辑你的默认出生信息 |
| API Key | 填入 DeepSeek API Key 以启用 AI 增强解读 |
| 主题 | 切换「深邃夜空」或「紫水晶」两种暗黑主题 |
| 关于 | 版本号、技术栈、隐私说明 |

### 获取 DeepSeek API Key

1. 访问 [platform.deepseek.com](https://platform.deepseek.com/)
2. 注册并充值（最低 ¥1）
3. 在「API Keys」页面创建 Key
4. 复制粘贴到设置页的 API Key 输入框

---

## 五、隐私与数据

- **所有数据仅存储在浏览器的 localStorage 中**
- 系统没有后端服务器，没有人能看到你的数据
- 清除浏览器缓存会清空所有本地数据
- 如需迁移数据，可导出 localStorage 中的 `user` 和 `history` 两个 key

---

## 六、注意事项

- 占星计算精度经 2400+ 测试用例验证，行星位置误差 < 1°
- 塔罗解读基于 Rider-Waite 体系，仅供娱乐和参考
- AI 增强解读由 DeepSeek 生成，内容不代表本工具立场
- 宫位制默认使用 **Placidus**（普拉西度），当前版本不支持切换
